# Receipts (optional supplement)

`/subs` pulls subscription receipts from your **Gmail connector** automatically —
you usually don't need to put anything here.

Use this folder only to **supplement** Gmail: drop exported `.eml`, `.txt`, or
`.html` receipts for subscriptions billed to an account that isn't your Gmail
(e.g. a work address, or a PDF invoice). The skill reads these in addition to the
Gmail search.

If the Gmail connector isn't enabled, `/subs` will ask you to enable it
(Connectors → Gmail) — that's the primary receipt source.
